# bot-protection

## Description

Cloudflare Turnstile challenge widget embedded on high-risk public forms
(signup, login, password reset, contact). Turnstile is privacy-respecting (no
visible CAPTCHA puzzle), free, and validates server-side via a Cloudflare API
call in the action. The secret key is stored as a Wrangler secret.
