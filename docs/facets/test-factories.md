# test-factories

## Description

Typed factory functions and database seed scripts for generating realistic test
data. Factories (`createUser`, `createTeam`, `createSubscription`, etc.) accept
partial overrides and wire up relationships automatically. A `db:seed` npm
script populates a local D1 instance with a representative dataset for manual
and E2E testing.
